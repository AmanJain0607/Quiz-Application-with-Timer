# Quiz Application with Timer

A timed multiple-choice quiz application in Java.

## Features
- Question bank with multiple-choice answers
- 10-second timer per question
- Score calculation
- Results summary

## How to Run
1. Compile: `javac QuizApp.java`
2. Execute: `java QuizApp`
